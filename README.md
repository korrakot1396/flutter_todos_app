## Project installation process

_For server_

```shell
clone https://github.com/korrakot1396/golang-gin-todos.git from https://github.com/korrakot1396/golang-gin-todos
cd golang-gin-todos
go mod download
go mod tidy
go run cmd/main.go
```

_For unit test_

```shell
cd golang-gin-todos
cd ./tests
go test
```
